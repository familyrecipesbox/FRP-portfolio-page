Development

In this project, we used the following tools and technologies to develop the Family Recipes Box App:

GitHub

We used GitHub coding repository, source code control, and project/task management.

Firebase

We used Firebase for database, file storage, authentication, and web hosting

Google Maps and Google Places API

We used Google Maps API to display user's current geo-location and Google Places

Bootstrap CSS

Because of the limited timeframe, we relied on Bootsrap CSS for front-end development.

Sweet Alerts CSS

Because of the limited timeframe, we relied on Sweet Alerts CSS for stylied prompts and alerts.

Moment JS

We used Moment JS for quick conversion of the UNIX timestamp to the date format.

Fontawesome CSS

To save time on UI asset development, we used Fontawesome CSS for UI icons and other UI elements.
